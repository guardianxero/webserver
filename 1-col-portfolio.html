Tab-oo Chrome Extension

A product of a sleepless night at the Facebook Hackathon

Made at the 2013 Annual NorCal Facebook Hackathon with my friends and fellow EECS majors Kunal Roy, Chris Dock, and Eric Wu, this is a Chrome extension that combines an automatic tab sorter with a notification system for active tabs. All four of us had always had issues with having too many tabs open at once in Google Chrome, so we designed this extension in hopes of simplifying our experience in Google Chrome. The current version on the Chrome store is a prototype of the final product, and there is still much polishing to be done. The extension was written in Javascript and implemented both Jquery and Chrome APIs.

Personal Movie Server

A Result of Boring Summer Days

Before coming to Berkeley, I partnered with my friend Manu Goyal to design a HTTP server in Google Go that would allow us to directly download movies from the home directories of the server. Much of what I did was help my friend debug the code, as he was the much more experienced coder, but I helped with the server.go file and wrote the CheckAccessHandler that runs the login.html page. Overall, it was a good learning experience for me to understand more of how an HTTP server works.
